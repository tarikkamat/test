iyzico ile WooCommerce mağazanızda abonelik ödemelerini güvenli bir şekilde alın.

== Description ==
iyzico altyapısı ile kredi kartı ve banka kartı ile güvenli abonelik ödemeleri.

== Installation ==
1. Yükleyin ve etkinleştirin.
2. WooCommerce > Ayarlar > Ödemeler > iyzico Abonelik üzerinden yapılandırın.
